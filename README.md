## LazyVim Full-Stack Setup (Python Included)

This repository provides a **LazyVim-based development environment** for those who want a full-stack setup with Python integration.

I previously used VS Code for over two years, but it never gave me a sense of full control. The increasing AI bloat became distracting. This setup focuses on a different approach: **minimal, fast, and fully controlled development**.

There is no heavy AI integration here. You get **raw coding power directly in your hands**. That said, completely ignoring AI is unrealistic, so I included a lightweight AI plugin for small suggestions—nothing intrusive.

This configuration aims to give you:

- A clean and efficient Neovim experience
- Full-stack development capability
- Python integration out of the box
- Minimal distractions with optional AI assistance
- An IDE-like workflow in a lightweight environment
